Frequently Asked Questions

  1. What kind of jobs can I run on Hungry Hippos platform?
  2. What is the maximum size of data for which I can submit jobs for, using this tester web application?
  3. Where should I put my data?
  4. What all file formats are supported?
  5. Can I submit multiple jobs in the one go?

Answers

  1. What kind of jobs can I run on Hungry Hippos platform?

    You can run almost all jobs that you can run on Hadoop platform.

  2. What is the maximum size of data for which I can submit jobs for, using this tester web application?

    It really depends on RAM and CPU requirement of job/computation you are trying to perform. But as this tool is made available for testing purposes, you can run jobs of maximum data size of 30 GB and computations that can be accommodated in 2GB RAM on one node. This limitation is because when you submit a job, we create a 10 nodes cluster with each node having 2GB RAM and 40 GB hard disk.

    We have successfully tested computations on data of size of 10GB for sum and median metric calculations.

  3. Where should I put my data?

    Data should be available and accessible over internet.

  4. What all file formats are supported?

    As of now only CSV file format is supported.

  5. Can I submit multiple jobs in the one go?

    Yes. You can submit a job matrix which contains more than one job/computation.
